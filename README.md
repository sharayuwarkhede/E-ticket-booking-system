This is the miniproject on topic E-tickect booking system 
A console-based E-Ticket Booking System developed using Java. This project allows users to book tickets for Bus, Train, and Airway transportation with features
like passenger management, seat booking, fare calculation, discounts, add-on services, ticket summary, and ticket cancellation.
This project is developed for educational and learning purposes.
Developed By First Year Computer Science Engineering Student
---
# Features #

-  Multiple Passenger Booking
-  Journey Date and Time Selection
-  Destination Selection
-  Multiple Transportation Modes
  - Bus
  - Train
  - Airway
-  Interactive Seat Booking
-  Automatic Fare Calculation
-  Discount System
  - Female Discount
  - Student Discount
  - Senior Citizen Discount
  - Round Trip Discount
  - Group Booking Discount
-  Add-on Services
  - Extra Luggage
  - Meal
  - Window Seat
  - Insurance
  - WiFi
-  Ticket Summary
-  Ticket Cancellation

#  Project Workflow #

1. Enter number of passengers.
2. Select journey date and time.
3. Enter passenger details.
4. Select destination.
5. Choose available mode of transportation.
6. Book seats.
7. Select optional add-on services.
8. View ticket summary.
9. Cancel ticket (optional).
